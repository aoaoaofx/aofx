# Jd-Pruchase-Kill

### 扫码登陆
 
 Start.class是程序入口,exe运行必须要有jre1.8运行环境
 
 **配置文件initData.txt文件必须与.exe文件或者项目在同一目录下**
 
### 需要设置
 
     商品id pid;
     
     eid = "";
     
     fp = "";

商品id为商品详情url中的一串数字

eid以及fp在订单结算页面 https://trade.jd.com/shopping/order/getOrderInfo.action

F12打开console，输入_JdTdudfp

或者F12打开Sources，添加_JdTdudfp参数
